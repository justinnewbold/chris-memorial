# Chris Newbold Memorial

A beautiful memorial tribute site for Chris Francis Newbold (1943-2020).

## Live Site
[chris.newbold.cloud](https://chris.newbold.cloud)

## Features
- Auto-playing tribute video
- Elegant, respectful design
- Mobile responsive
- Click to unmute audio

## Setup
Upload your `tribute.mp4` video file to this repository and the site will automatically deploy.

## Video Requirements
- Format: MP4 (H.264 codec recommended)
- The video must be named exactly `tribute.mp4`
